# Portfolio Management API

This project is a Portfolio Management REST API built with Spring Boot.

Users can:
- register and login,
- manage their portfolios,
- add and update education, experience, project, and skill information.

Technologies used in the project:

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Redis Cache
- Liquibase
- Docker
- Maven

Main features:
- Authentication and authorization with JWT
- Redis caching
- Global exception handling
- Validation
- Layered architecture
- REST API principles

The project was tested using Postman.
